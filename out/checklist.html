✅ Moonlust Checklist

  • SEO & Social Sharing: ✅ - Đã cấu hình chuẩn SEO: title, desc, keywords, openGraph, Twitter card. Tích hợp global component SeoHead.
  • Đa ngôn ngữ (i18n): ✅ - Đã có dynamic language switch + auto detect IP + thư mục dịch chuẩn hoá vi/en/ja.
  • Trang chủ (Homepage): ✅ - Hiển thị 3 truyện nổi bật (Card grid), có ảnh + tiêu đề + mô tả + trạng thái. Responsive đẹp.
  • Trang chi tiết truyện: ✅ - Đã có slug động, hiển thị 3 chương + ảnh đầu mỗi chương + đa ngôn ngữ nội dung.
  • Giao diện hình ảnh: ✅ - Đã gắn ảnh logo truyện, ảnh minh họa chương, bo góc, shadow đẹp, dùng CDN path chuẩn.
  • Phân quyền & User: ❌ - Chưa tích hợp chức năng login / register (đang chờ Firebase/Auth0).
  • Bình luận & đánh giá: ❌ - Chưa tích hợp hệ thống comment hoặc rating. Sẽ gắn sau khi có user auth.
  • Lưu lịch sử đọc: ✅ - Đã dùng localStorage ghi nhớ truyện vừa đọc, chuẩn bị sẵn để gắn DB sau.
  • Quản trị truyện (CMS): ❌ - Chưa triển khai admin UI/backend CMS, nhưng đã dự trù sẽ dùng Next Admin hoặc Headless CMS.
  • Quảng cáo / Hợp tác: ✅ - Đã có AdContactSection với nút liên hệ qua email + khối giao diện gọi CTA mạnh.
